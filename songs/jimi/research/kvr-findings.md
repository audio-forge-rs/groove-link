# KVR Audio Research Findings

## CLA Guitars by Waves - KVR Product Listing

**URL**: https://www.kvraudio.com/product/cla-guitars-by-waves

### Technical Specifications
- **Type**: Virtual Effect Audio Plugin
- **Formats**: VST, VST3, Audio Units (AU), AAX
- **Platforms**: macOS, Windows
- **Developer**: Waves Audio
- **Part of**: Chris Lord-Alge Artist Signature Series (6 plugins total)

### CLA Signature Series Bundle
The full series includes:
1. **CLA Bass** - Bass processing
2. **CLA Drums** - Drum processing
3. **CLA Vocals** - Vocal processing
4. **CLA Guitars** - Guitar processing
5. **CLA Effects** - General effects processing
6. **CLA Unplugged** - Acoustic/unplugged instrument processing

### KVR Forum User Opinions on CLA Guitars

**From "Waves CLA Effects, your thoughts" thread (t=502929)**:

Positive opinions:
- Users find CLA Guitars "very useful" for quick guitar mixing
- The guitar plug-in has "a good doubler that doesn't sound flangey"
- Works as "a quick and easy way to get a not too bad sound" for recording guitars
- Presets speed up mixing workflow significantly

Mixed/Negative opinions:
- "Really wanted to like the CLA bundle, but for what I do generally it just didn't fit the bill"
- Limited to rock/modern rock genres
- Not enough control for experienced engineers
- Some users prefer more surgical tools

**General consensus**: CLA Guitars is valued for speed and convenience in rock mixing contexts, but lacks the flexibility needed for other genres or detailed sound design.

## Bitwig Guitar Processing - KVR Forum Discussions

### "A praise for the AMP tool" (t=487233)

Users describe Bitwig's Amp device as:
- "One sweet sounding software amp"
- "Responds really well on the gain level from the guitar"
- Works well combined with distortion plugins
- Can be used on non-guitar sources (snare drums, synths) for character
- Cabinet models add harmonics: some models amplify odd harmonics, others even harmonics
- Useful for "taming peaks" on various sources

### "Recording Guitar Effects live into Bitwig" (t=459108)

Technical discussion about:
- Setting up audio routing for live guitar processing
- Using Bitwig's effects chain for real-time guitar FX
- Selecting guitar audio outputs as track sources

### "Amp Device's Cabinet models are duplicates" (t=541372)

Issue raised about Bitwig's Amp cabinet models, suggesting some modes may produce very similar results despite different labels.

### Guitar Plugin Alternatives on KVR

Users recommend alongside/instead of Bitwig's built-in Amp:
- **ToneLib GFX**: "Probably indeed the best native guitar amp plugin that will work in Bitwig"
- **Amplitube 4**: Full guitar rig simulation
- **Guitar Rig**: Comprehensive effects and amp modeling
- **Neural DSP**: Modern amp modeling (works as VST in Bitwig)

### "Saturator - tips, thoughts, ideas" (t=544868)

Discussion about creative uses of Bitwig's Saturator:
- Can be used for subtle warmth or aggressive overdrive
- Log-domain processing gives "ear-like" response
- Curve editor allows precise shaping of saturation character
- Skew controls for asymmetric clipping (tube-like even harmonics)
- Useful for guitar overdrive emulation when combined with EQ

### "Override or similar effect" (t=434446)

Users looking for guitar-style override/overdrive effects:
- Bitwig's built-in distortion options discussed
- Saturator recommended for more musical overdrive tones
- Third-party VST plugins suggested for authentic amp tones

### "using expression pedal in Bitwig with Guitar Rig" (t=491396)

Discussion about hardware integration:
- Expression pedal mapping to Bitwig parameters
- Using MIDI CC to control effect parameters in real-time
- Bitwig's modulation system praised for flexible parameter control

## Key Takeaways from KVR Research

### CLA Guitars
1. Best for rock/modern rock mixing, not a universal tool
2. The stereo doubling/pitch effect is its most unique feature
3. Works well as a "finishing touch" on already-good guitar recordings
4. Not a replacement for dedicated amp sims
5. Speed and convenience are its primary value propositions

### Bitwig for Guitar
1. The built-in Amp device is underrated but has limitations
2. Saturator is the key device for overdrive/tube emulation
3. The modulation system (38+ modulators) gives Bitwig an edge for creative guitar FX
4. Cabinet simulation in Amp is functional but not best-in-class
5. Most serious guitarists supplement with third-party amp sims
6. Bitwig's strength is in creative effects chains, not traditional amp modeling
7. FX chains within devices (reverb tank FX, delay feedback FX) enable unique guitar textures not possible with simple pedal chains
