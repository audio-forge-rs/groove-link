## Key Concepts Summary

### Two Sequencers
Bitwig has both an **Arranger** (linear timeline) and **Clip Launcher** (non-linear triggering). They can work together or independently.

### Devices & Modulators
Every track has a device chain. Devices can be nested (containers). Any parameter can be modulated by modulators (LFOs, envelopes, etc.).

### The Grid
Bitwig's modular synthesis environment. Create custom instruments and effects by connecting modules.

### Expressions
Per-note and per-event data beyond just velocity: gain, pan, pitch, timbre, pressure, chance.

### Operators
Transform note sequences with probability (Chance), repetition (Repeats), conditional playback (Occurrence, Recurrence).
